# AI Productivity Assistant

## Overview
This project is an AI-powered web application that helps users manage tasks, notes, and schedules through a conversational interface.

## Features
- Real-time AI responses
- Task and note assistance
- Scheduling help
- Clean chatbot UI
- Cloud deployment (24×7 access)

## Tech Stack
- Python
- Streamlit
- OpenAI API
- Google Cloud Run

## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py

## Deployment
Deployed using Google Cloud Run for scalable and continuous availability.
